Unit 1 Notes:

Outcomes:
1)Appreciate the value of data visualisation.
2)Appreciate the basic principles of data visualisation.
3)Know about different tools and techniques used for data visualisation.

The lecture cast Principles of Data Visualisation was insightful. I completed the formative activity as well as the case study which was posted on the Module Wiki. This tapped into communication with peers on the Wiki as well as easing into data visualisations and the integral role it can play. View more details.

Unit 2 Notes:

Outcomes:
1)Critically evaluate different data visualisation principles used.
2)Identify and apply data visualisation grammar more effectively.

The lecture cast Principles of Data Visualisation 2 was also helpful in better understanding concepts. The formative activity for this week was a collaborative discussion on the Report on Ryanair and Easyjet Websites. My response can be found here: here.
Referencing and actively engaging with both websites and the various functions, strengths, weaknesses, visible features etc. were exercised during this activity.

Unit 3 Notes:

Outcomes:
1)Write and execute basic R script through Codio IDE.
2)Understanding of dataframe and basic visualisation techniques.
3)Understanding of libraries and basic functions.

In this module, I started exploring Rstudio. The challenge was installing it and getting comfortable with the functionality and layout. I did the recommended reading for this module and the exercises from the textbook, which helped ease the nerves. I did not partake in the peer response requirement for this module on the previous formative activity. None of my peers had commenced or initiated this exercise. Reflecting on this, I could have shown leadership and completed this exercise irrespective in anyone else did.
